# BookDiary
## 책 검색 및 기록 App
> 1. 다음 책 검색 api 활용 검색 및 저장 앱
> 2. EventBus, Paging3 등 라이브러리 사용
> 4. 개발 플랫폼:  Android Studio Dolphin, API Level 33
------------
### 개발환경
+ Android Studio Dolphin  
+ minSdkVersion : 22  
+ targetSdkVersion : 33 
+ Language : Kotlin 
------------
### Skill & Library
+ Materail3 UI/UX 
+ dataBinding사용
+ Room DB 
+ Paging3 
+ DI Hilt   
+ OkHttp4/Retrofit 활용 Coroutine을 이용한 REST 처리 
------------
### 사용 API
+ kakao REST API
------------  
### 주요기능
+ 책 검색  
+ 원하는 책 저장
